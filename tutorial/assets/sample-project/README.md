# Pets Diagram Project

Sample project for the Mermaid Hybrid Stack Tutorial.

These files are reference copies used across the 7 tutorial exercises. Copy any file you
need into your working project directory (`~/projects/pets-diagrams`).

## Files

| File | Exercise | Description |
|------|----------|-------------|
| `pets.mmd` | 01, 02 | Pet store hierarchy flowchart (light theme) |
| `pets-dark.mmd` | 02 | Same flowchart, rendered with dark theme |
| `adoption-stats.vl.json` | 06 | Monthly adoption statistics bar chart (Vega-Lite) |
| `shelter-architecture.d2` | 06 | Pet shelter system architecture diagram (D2) |
